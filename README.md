# ExponentialSums
Calculation and drawing of John D. Cook's [Exponentional Sum of the Day](https://www.johndcook.com/expsum/). Please visit his website!

```math
    \sum_{n=0}^m
        e^{\left( 2\pi i \sum_{i=1}^k \frac{n^i}{n_i} \right)}
```
